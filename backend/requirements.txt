# AI Meeting Assistant — Backend
# ====================================
# FastAPI server for Arabic audio transcription and campaign insight extraction.
#
# Architecture:
#   1. Receives .webm audio file from the Chrome Extension
#   2. Transcribes Arabic speech via Groq API (Whisper-large-v3)
#   3. Extracts summary, action items, and campaign budgets via Groq (Llama-3)
#   4. Returns structured JSON for downstream consumption (n8n → ClickUp/Slack)
